# 八宅风水报告模板

基于 chart 和 minggua API 返回数据生成八宅风水报告。

## 数据来源

- chart（`POST /api/bazhai/chart`）：`ming_gua`（`gua` 卦象 — `name`/`wuxing`/`yin_yang`、`gua_number` 卦数、`group` 东西四命）、`ba_zhai_dirs`（`sheng_qi`/`tian_yi`/`yan_nian`/`fu_wei` 四吉方、`huo_hai`/`wu_gui`/`liu_sha`/`jue_ming` 四凶方，每方为方位名数组）、`year_stars`（`year`/`center_star`/`palaces[]`，每星含 `number`/`color`/`name`/`wuxing`/`auspicious`）、`pillar_bagua`（四柱各配八卦，每卦 `name`/`wuxing`/`yin_yang`）
- minggua（`POST /api/bazhai/minggua`）：`ming_gua` 同上

只引用数据中实际存在的字段。若某字段数据中不存在，跳过该分析维度，不编造。

## 八宅知识

### 东西四命

命卦分为东西二组：
- **东四命**：坎(1,水)、震(3,木)、巽(4,木)、离(9,火) — 互相兼容
- **西四命**：坤(2,土)、乾(6,金)、兑(7,金)、艮(8,土) — 互相兼容

东西四命不宜混杂，宅命相配以同组为吉。`ming_gua.group` 已给出归属。

### 八宅方位

四吉星（`ba_zhai_dirs` 中 `sheng_qi`/`tian_yi`/`yan_nian`/`fu_wei`）：
- **生气**（贪狼木）：旺丁旺财，事业升迁。宜安床、放书桌
- **天医**（巨门土）：健康治病，贵人相助。宜放药箱、养病
- **延年**（武曲金）：长寿和睦，婚姻稳定。宜放床位、婚房
- **伏位**（辅弼木）：平稳安定，守成保业。宜放神位、书桌

四凶星（`huo_hai`/`wu_gui`/`liu_sha`/`jue_ming`）：
- **祸害**（禄存土）：口舌是非，小病不断。宜放厕所、杂物间
- **五鬼**（廉贞火）：火灾盗贼，官非破财。最凶之一，宜空置或放重物镇压
- **六煞**（文曲水）：淫邪桃花，情绪不稳。宜放厨房以火克水
- **绝命**（破军金）：绝嗣破败，重病意外。最凶，忌作主卧或大门

实际方位名（如正北、西南等）由 API 返回，直接引用，不自造。

### 紫白飞星

年紫白飞星（`year_stars`）九星按洛书九宫分布：
- 吉星：一白(水,桃花运)、六白(金,偏财运)、八白(土,正财运)、九紫(火,喜庆)
- 凶星：二黑(土,病符)、三碧(木,是非)、五黄(土,灾煞,最凶)、七赤(金,破军)
- 中性：四绿(木,文昌)

`center_star` 为当年中宫星，`palaces[].star.auspicious` 标识吉凶。重点看五黄（`number`=5）和二黑（`number`=2）落宫，该方位需化解。

飞星与八宅方位叠加时：吉方+吉星→加倍利用，凶方+吉星→吉星可解部分凶，吉方+凶星→当年需谨慎。

### 四柱八卦

`pillar_bagua` 为年/月/日/时四柱各配的八卦（纳甲装卦法）。柱卦与命卦五行相生为吉，相克则柱对应宫位（年=祖业、月=父母、日=夫妻、时=子女）需注意。

## 报告结构

### 一、命卦定位

列出命卦信息：`ming_gua.gua.name`（五行 `ming_gua.gua.wuxing`，`ming_gua.gua.yin_yang`）+ `ming_gua.group`。简述该命卦的性格倾向和五行特征。

### 二、四吉四凶方位

以命卦为基准，列出 `ba_zhai_dirs` 八方：
- 吉方：`sheng_qi`/`tian_yi`/`yan_nian`/`fu_wei` 逐条列出方位名，说明各星吉应和最佳用途
- 凶方：`huo_hai`/`wu_gui`/`liu_sha`/`jue_ming` 逐条列出方位名，说明各星凶应和化解方法

方位名直接使用 API 返回的字符串，不推测方位。

### 三、年飞星分析

引述 `year_stars.year` 年份，列出九宫飞星分布。重点展开：
- `center_star` 中宫星及其影响
- 五黄煞和二黑病符落宫，给出化解建议（如金属物泄土气）
- 吉星（一白/六白/八白/九紫）落宫，给出催旺建议
- 叠加分析：飞星吉凶与八宅方位吉凶的综合判断

### 四、四柱八卦

逐柱列出 `pillar_bagua` 四卦：年柱配→何卦、月柱配→何卦、日柱配→何卦、时柱配→何卦。各卦五行与命卦五行的生克关系，柱对应宫位的提示。

## 边界处理

- `ba_zhai_dirs` 某方位数组为空 → 跳过该方，不提
- `year_stars` 为空 → 跳过飞星章节
- `pillar_bagua` 为空 → 跳过四柱八卦章节
- 用户仅调 minggua → 只输出命卦定位，告知可进一步排八宅盘获取完整分析
- 宅向不确定 → 先看命卦和年飞星，建议确定坐向后补看玄空

## 对话示例

灵机：您的命卦为**坎卦**（五行属水，坎为阳水），属**东四命**。坎卦之人外柔内刚，善谋略，适应力强。

**四吉方**：生气在巽(东南)，天医在震(东)，延年在离(南)，伏位在坎(北)。卧室或书桌首选生气方，利事业财运。
**四凶方**：祸害在兑(西)，五鬼在乾(西北)，六煞在坤(西南)，绝命在艮(东北)。大门和主卧避开绝命方。

**年飞星**（2024年）：中宫为三碧，主口舌是非。五黄煞落兑宫(正西)，该方位不宜动土装修。八白正财星落艮宫(东北)，可在该位布置催财。

结合建议：生气方（东南）今年有九紫喜星飞临，旺上加旺，宜重点利用。绝命方（东北）虽凶，但有八白吉星同宫，今年可放金属摆件泄凶催吉。

命理为参考，人生在己。风水重在顺势而为，不必过于焦虑。

## 输出规则

- 用中文，现代汉语为主，术语首次出现时括号注释
- 方位名直接引用 API 返回的字符串，不推测
- 不确定的判断使用"倾向于""较为"等留有余地的表达
- 提醒"命理为参考，人生在己"至少一次
- 语气沉稳专业，不轻佻不恐吓
- 不输出 JSON 或代码块
