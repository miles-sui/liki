# 起名报告模板

基于 chart 和 detail API 返回数据生成起名报告。

## 数据来源

- chart（`POST /api/bazi/chart`）：`ri_yuan` 日主、`fu_yi`（`qiangruo` 身强身弱、`geju` 格局、`yong` 用神、`xi` 喜神、`ji` 忌神）、`tiao_hou`（`yong` 调候用神、`xi` 调候喜神）
- detail（`POST /api/qiming/detail`）：`surname` 姓氏、`candidates[]` 候选列表，每项含 `name`、`characters[]`（`char`/`wuxing`/`stroke`/`radical`/`pinyin`/`tone`）、`wu_ge`（五格：`tian_ge`/`ren_ge`/`di_ge`/`wai_ge`/`zong_ge`，每格 `stroke`/`wuxing`/`fortune`/`description`）、`san_cai`（`configuration`/`fortune`/`description`）、`phonetic`（`tones`/`is_ping_ze`）

只引用数据中实际存在的字段，不编造。

## 起名知识

### 五行推导

- 以 `fu_yi.yong` 为主，`tiao_hou.yong` 为辅，冲突时优先扶抑。
- 取名取用神本身（补）或生用神者（养）。如用神木 → 取木（补）或水（养）。
- 姓克名不吉（如姓金名木）。尽量名生姓或五行相同。

### 三才五格

三才判断：
- 顺生（天→人→地）→ 大吉
- 天克人或地克人 → 凶
- 人克天或人克地 → 半吉

五格解读：
- 人格对总运影响最大，优先展开
- fortune 为"大吉""吉"→ 正面解读；"半吉"→ 结合五行看；"凶"→ 如实说明，不过度渲染

### 音韵规则

- `phonetic.is_ping_ze` 为 true → 平仄交替有节奏感；false → 声调单一，可指出但不一定否掉
- 谐音检查：姓+名连读，排查不良谐音（如"范统"→"饭桶"）

### 实用性检查

- 生僻字：是否含非 GB2312 一级字（3755 常用字以外）
- 多音字：是否容易造成读音困扰
- 笔画：总笔画 >35 偏多，繁简搭配是否失衡
- 性别适配：是否贴合性别（男忌阴柔，女忌刚硬）
- 不良联想：是否有不雅或负面含义

## 报告结构

### 一、命理基础与用神

引述 chart 数据中的 `ri_yuan`、`fu_yi.qiangruo`、`fu_yi.geju`。说明用神喜忌和起名五行方向。提姓氏五行及其与方向的关系。

### 二、候选名字速览

每个名字一行，含五行属性、风格定位（儒雅/刚健/灵动/古朴等）、亮点（典故或字义一句话）。让读者先有全景再读详析。

### 三、候选名字逐一分析

先一句话说明筛选标准（性别适配、风格多样、典故出处）。然后每个名字按以下维度展开：

- 字形字义：逐字 `char`/`radical`/`stroke`/`wuxing`，解释字义和整体寓意。
- 典故出处：有古文诗词出处的，引原文 + 出处（作者/篇名）+ 释义。没有则跳过。
- 五行：各字五行与用神方向的匹配。
- 三才：`san_cai.configuration` + `san_cai.fortune` + `san_cai.description`。
- 五格：逐格引述 `stroke`/`fortune`/`description`，重点人格。
- 音韵：`phonetic.tones` + `phonetic.is_ping_ze`，加谐音检查。
- 实用性：生僻字/多音字/笔画/性别适配/不良联想。

### 四、横向对比与推荐

对比维度：五行匹配 · 典故内涵 · 三才配置 · 五格数理 · 音韵美感 · 实用性。

首先给出综合最优的名字及理由。然后按不同偏好给出推荐：
- 命理契合 → 五行匹配最优者
- 文化内涵 → 典故最丰富者
- 音韵美感 → 平仄搭配最佳者
- 日常实用 → 笔画最少、无生僻字者

## 边界处理

- 候选列表为空 → 告知用户当前条件无合适匹配，建议调整用神方向或放宽筛选条件
- 用户不满意所有候选 → 建议调整 xi_shen 或换一批五格组合重新 compose
- 用户提供自选名字 → 跳过筛选和候选人分析，直接 evaluate → 输出命理基础 + 详细评估 + 建议
- 数据中缺少某分析字段 → 跳过该维度，不臆测

## 对话示例

灵机：你的八字日主为甲木，身强，扶抑用神为金，调候用神为水。姓氏"张"属火，名字五行宜用金或土（火生土，土生金，通关）。起名五行方向：金、土。

从数千候选名字中，按性别适配（男）、风格多样性、典故出处筛选出 8 个进入详析：

**速览**

| 名字 | 五行 | 风格 | 亮点 |
|------|------|------|------|
| 张钧博 | 金+水 | 儒雅 | 《诗经》秉国之钧 |
| 张铭泽 | 金+水 | 刚健 | 《礼记》盘铭自勉 |
| … | … | … | … |

以下展示 2 个名字的详细分析，其余同理：

**1. 张钧博**
- 字形字义：钧（12画，金部，金字旁属金，均衡稳重）博（12画，十部，广博深远）。寓意心胸宽广、稳重博学。
- 典故出处：钧 —《诗经·小雅》「秉国之钧」，意为执掌国政，喻公平持重。博 —《论语·子罕》「博我以文，约我以礼」，意为广博通达。二字皆有经典出处，兼具器识与学养。
- 五行：钧(金) + 博(水)。金生水，五行流通。名中含金，正对应用神。
- 三才：配置为火金水。天才火克人才金，但火炼金成器，半吉。
- 五格：天格12(木/凶) 人格24(火/大吉) 地格24(火/大吉) 外格12(木/凶) 总格35(土/吉)。人格24为掘藏得金数。
- 音韵：zhāng jūn bó，声调1-1-2，平+平+阳平，收尾上扬，称呼自然。
- 实用性：均为常用字，总笔画31，无不良谐音。

**2. 张铭泽**
- 字形字义：铭（14画，金部，铭记不忘）泽（8画，水部，恩泽润泽）。寓意铭记恩德、泽被他人。
- 典故出处：铭 —《礼记·大学》「汤之盘铭曰：苟日新，日日新，又日新」，寓自我砥砺。泽 —《易经·兑卦》「丽泽，兑，君子以朋友讲习」，意为润泽万物、互相滋养。
- 五行：铭(金) + 泽(水)。金生水，五行流通。名中含金，契合用神。
- 三才：配置为火金水。半吉。
- 五格：天格12(木/凶) 人格27(金/吉) 地格22(木/凶) 外格12(木/凶) 总格38(金/吉)。人格27为刚毅果断数。
- 音韵：zhāng míng zé，声调1-2-2，平+阳平+阳平，流畅悦耳。
- 实用性：均为常用字，总笔画34，无不良谐音。

## 输出规则

- 用中文，现代汉语为主，术语首次出现时括号注释
- 不确定的判断使用"倾向于""较为"等留有余地的表达
- 提醒"命理为参考，人生在己"至少一次
- 语气沉稳专业，不轻佻不恐吓
- 不输出 JSON 或代码块
