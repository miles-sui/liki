# 紫微斗数报告模板

基于 chart、daxian、liunian、liuyue、liuri、bond API 返回数据生成紫微斗数报告。

## 数据来源

- chart（`POST /api/ziwei/chart`）：
  - `palaces[]` — 十二宫。每宫含 `index`/`name`/`gan`/`zhi`/`is_body_palace`/`stars[]`。
  - `stars[]` — 星曜。每星含 `star`(索引)/`name`/`is_major`/`si_hua`(有则禄权科忌)/`brightness`(庙旺利平陷)。
  - `ming_gong` — 命宫索引（恒为0）、`shen_gong` — 身宫索引、`ju_shu`/`ju_shu_name` — 局数、`ziwei_pos` — 紫微星宫位。
  - `si_hua` — 本命四化（starIndex→禄权科忌）、`year_gan`/`hour_zhi` — 年干/时支、`birth_year`/`gender`。
  - `patterns[]` — 格局。每项 `name`/`description`/`score`（0下格、1中格、2上格）。
- daxian（`POST /api/ziwei/daxian`）：`[{start_age, end_age, palace, name}]`，12项，每项为十年大限。
- liunian（`POST /api/ziwei/liunian`）：`ming_gong`/`ming_gong_name`、`si_hua`（流年四化）、`si_hua_palace`（各化星落宫）、`minor_stars`（流年辅星位置）。
- liuyue（`POST /api/ziwei/liuyue`）：`ming_gong`/`ming_gong_name`/`si_hua`。
- liuri（`POST /api/ziwei/liuri`）：`ming_gong`/`ming_gong_name`/`si_hua`。
- bond（`POST /api/ziwei/bond`）：`a_into_b`/`b_into_a`（双方命宫互入）、`star_cross[]`（`star`/`from_a`/`into_b`）、`sihua_cross[]`（`star`/`type`/`into_b`）。

只引用数据中实际存在的字段。若某字段数据中不存在，跳过该分析维度，不编造。

## 紫微知识

### 十二宫

| 宫位 | 主事 | 解读重心 |
|------|------|----------|
| 命宫 | 性格、天赋、一生运势基调 | 星曜组合定命格高低 |
| 兄弟 | 手足关系、同辈助力 | 主星吉凶看兄弟姐妹缘 |
| 夫妻 | 配偶品性、婚姻质量 | 星曜定配偶类型，四化看感情变数 |
| 子女 | 子嗣、晚辈、下属 | 主星看子女多寡和缘分 |
| 财帛 | 赚钱能力、理财方式 | 星曜看财富来源和进出模式 |
| 疾厄 | 健康、体质、疾病倾向 | 煞星多则需注意对应脏腑 |
| 迁移 | 外出运、社会形象 | 对外表现，与命宫互为表里 |
| 交友 | 朋友、合作伙伴 | 仆役宫，看人际关系质量 |
| 官禄 | 事业、学业、社会地位 | 与命宫配合看人生成就 |
| 田宅 | 房产、家庭、祖业 | 不动产运和家庭和睦度 |
| 福德 | 精神世界、享福能力 | 内心满足感和晚年福分 |
| 父母 | 父母、长辈、上司 | 与权威的关系和助力 |

命宫为全盘枢纽。身宫（`shen_gong`）主后天发展重心，身宫所在宫位为中年后的重点领域。

### 十四主星

**紫微系（六颗）**：
- 紫微：帝王星，尊贵权威，领导力。喜得左辅右弼拱照
- 天机：智谋星，聪明善变，策划力。变动多，不宜独坐
- 太阳：光明星，热情外向，贵气。旺则慷慨，陷则劳碌
- 武曲：财星，刚毅果断，执行力。掌钱财，略带孤克
- 天同：福星，温和享福，人缘好。懒散倾向，需煞星激发
- 廉贞：次桃花，复杂多变。能文能武，化忌则易陷是非

**天府系（八颗）**：
- 天府：库星，包容稳重，理财力。保守有余开创不足
- 太阴：月星，细腻温柔，内秀。旺则端庄，陷则阴柔
- 贪狼：正桃花，多才多艺，交际力。欲望强，需制化
- 巨门：暗星，口才分析，善辩论。化忌则口舌是非
- 天相：印星，辅助公正，协调力。依赖性强，宜配强主星
- 天梁：荫星，老成稳重，有寿。好管闲事，略带孤高
- 七杀：将星，刚猛敢拼，开创力。孤克，宜有吉星化解
- 破军：先锋星，破旧立新，冲动。变动大，不宜守成

### 辅星与煞星

- 吉辅：左辅(助)、右弼(助)、文昌(文才)、文曲(才艺)、天魁(贵人)、天钺(贵人)、禄存(财禄)、天马(奔波)
- 煞星：擎羊(刚暴)、陀罗(拖延)、火星(突发)、铃星(暗算)、地空(空想)、地劫(破耗)

辅星不单独定吉凶，搭配主星看增强或削弱。

### 四化

`si_hua` 为 starIndex → 化气的映射：
- **化禄**（禄）：增加、机会、财富、人缘、扩散。吉
- **化权**（权）：掌握、权力、控制、执行力。吉，但带压力
- **化科**（科）：名声、才华展现、贵人提携。吉，温和
- **化忌**（忌）：收敛、执着、烦恼、缺失。凶，但也是"在乎"的体现

四化是动态能量，生年四化（`chart.si_hua`）管一生基调，大限四化管十年，流年四化管当年。忌星所在宫位为人生课题所在，不必渲染恐吓。

### 庙旺利陷

`brightness` 影响星曜发挥程度：
- **庙**：星曜力量充分，吉星更吉、凶星减凶
- **旺**：力量次强，接近庙
- **利**：中等，星性平稳
- **平**：力量偏弱
- **陷**：吉星无力、凶星增凶

落陷的吉星不如庙旺的平星。庙陷需结合四化同看：庙+化禄锦上添花，陷+化忌雪上加霜。

### 局数与纳音

`ju_shu` 为五行局数（2水/3木/4金/5土/6火），由命宫纳音决定。纳音命局与命宫星曜五行相生为佳。命宫星克命局 → 根基略弱；命局生星 → 得局气滋养。

### 身宫

身宫（`shen_gong`）为后天发展重心：身宫在命→自我奋斗，在夫妻→婚姻为重，在财帛→财富驱动，在官禄→事业为重，在迁移→外出发展，在福德→精神追求。身宫所在宫位的星曜为中年后的运程重点。

## 报告结构

### 一、命盘总览

以表格列出十二宫（`palaces[]`）：宫名/宫干/宫支/身宫标记/主要星曜及亮度。概述：
- 命宫主星（`palaces[ming_gong].stars` 中 `is_major` 的星）及亮度
- 身宫所在（`shen_gong` 对应 `palaces[shen_gong].name`）及后天重心
- `ju_shu_name` 五行局数
- 三方四正（命宫/财帛/官禄/迁移）星曜是否有力

### 二、命宫详解

命宫为全盘核心，单独展开：
- 坐哪些主星（`is_major`）和辅星，各星 `brightness`
- 命宫之星有无 `si_hua`，若化禄/权/科/忌各有什么含义
- 命宫干支与五行局数的关系
- 紫微星位置（`ziwei_pos`）与命宫的关系（同宫/三合/对宫/六合等）
- 性格特质、天赋优势、注意事项

### 三、十二宫逐一解读

每宫（命宫已详述，从兄弟宫开始）依次分析：
- 主星 `name` + `brightness` → 该宫基本含义
- 辅星 → 补充或消减
- `si_hua` → 该宫若为化星所在，说明动态影响
- 吉星（左辅右弼天魁天钺文昌文曲禄存）加分
- 煞星（擎羊陀罗火星铃星地空地劫）需注意，结合主星看是否化解
- 重点宫位：夫妻（婚姻）、财帛（财运）、官禄（事业）、迁移（外出）

若篇幅过长，父母/兄弟/交友/田宅可概述，命宫/夫妻/财帛/官禄/迁移详述。

### 四、四化飞布

基于 `chart.si_hua`（生年四化）：
- 逐条列出哪星化禄/权/科/忌，落在哪个宫（查 `palaces[]` 中该星所在）
- 化禄 → 该宫领域有福气和机会
- 化忌 → 该宫领域为人生课题，需用心经营
- 四化之间的联动：化禄和化忌的宫位如何平衡

### 五、格局

若 `patterns[]` 非空：
- 列出各项 `name` + `description`
- `score` 为 2 → 上格，积极展开
- `score` 为 1 → 中格，有条件的吉格
- `score` 为 0 → 下格，如实描述但不过度渲染

常见格局理解：杀破狼主动荡开创、紫府同宫主富贵、日月并明主贵气、机月同梁主幕僚。若 `patterns` 为空，不提格局。

### 六、大限

基于 daxian：
- 列出 `[{start_age, end_age, name}]` 当前及未来 1-2 个大限
- 当前大限所在的宫位（`name`），该宫原局星曜在大限中被激活
- 大限宫位与本命四化的互动
- 关键节点：何时入强宫（命/官禄/财帛）、何时入弱宫（疾厄/交友）

### 七、流年

若数据含 `liunian`：
- 流年命宫 `ming_gong_name` + 该宫原局星曜
- 流年四化（`liunian.si_hua`）与生年四化的叠加效应
- 流年辅星（`minor_stars`）位置及影响
- 各化星落宫（`si_hua_palace`）的领域提示

若含 `liuyue`/`liuri` → 按"流月/流日命宫+四化"简要列出，不逐宫展开。

## 边界处理

- `palaces` 为空 → 无法生成报告，提示先排盘
- 某宫 `stars` 为空 → 说明该宫无主星（借对宫看），不编造星
- `patterns` 为空 → 跳过格局章节
- `si_hua` 为空 → 跳过四化章节
- daxian 数据为空 → 跳过大限章节
- 流年/流月/流日数据不含 → 跳过对应章节
- 命宫无主星（空宫）→ 借对宫论，说明"借迁移宫之X星安命宫"
- 某星 `brightness` 为空 → 不提亮度，只论星性
- 煞星集中某宫 → 如实指出但不恐吓，建议以吉星方位化解

| 兄弟 | 乙丑 | 天机(利) | |

命宫紫微庙旺坐命，属**金四局**。身宫在官禄宫。紫微为帝星，庙旺坐命得帝王气象，禀赋出众，有领导力和威严。迁移宫有七杀落陷，外出重实际挑战，不宜好高骛远。

**十二宫要点**：
- 夫妻宫太阳天梁同度，配偶正直稳重，有长上之德。太阳旺则配偶外向、心态阳光。
- 财帛宫武曲庙旺+禄存，武曲为正财星，禄存添福，财源稳定，适合理财和金融相关行业。
- 官禄宫天府坐守（身宫所在），事业以稳健积累见长，宜走管理路线。

**四化**：生年廉贞化禄在福德宫，精神享受丰足。破军化权在子女宫，对子女和下属管理有方。巨门化科在交友宫，朋友多为有才之人。天同化忌在田宅宫，家庭事务为人生课题，需多投入精力经营。

**格局**：`patterns[0].name` 为"紫府同宫"，`score` 为 2，上格。`description` 指出紫微天府同在命宫，君臣庆会，主富贵双全。

**大限**（当前）：34-43岁行迁移宫大限，七杀星被激活，此十年宜主动求变、开拓新局面，事业上或有跳槽或创业机会。

命理为参考，人生在己。紫微斗数揭示趋势，实际走向由你的选择和行动决定。

## 输出规则

- 用中文，现代汉语为主，术语首次出现时括号注释
- 十二宫表格用 markdown 展示
- 每条判断基于 API 返回数据，不编造星曜和宫位
- 星曜解读以 `brightness` 为依据，不假设庙旺
- 化忌如实说明，不渲染恐吓，强调"在乎所在"而非"注定的厄运"
- 不确定的判断使用"倾向于""较为"等留有余地的表达
- 提醒"命理为参考，人生在己"至少一次
- 语气沉稳专业，不轻佻不恐吓
- 不输出 JSON 或代码块
