# 紫微斗数 Engine — 算法理论设计

> Phase 0 参考文档。每个模块定义：输入值 → 查表/计算 → 输出值。无副作用、无 I/O。

## 约定

- 干支值域：天干 1–10（甲=1..癸=10），地支 1–12（子=1..亥=12）
- 天干归一如 `((x-1)%10+10)%10+1`，地支归一如 `((x-1)%12+12)%12+1`

---

## 0.1 十二宫定位

**输入**：`lunarMonth int`（1-12），`hourZhi Zhi`（1-12）

**公式**：

```
命宫地支：mingZhi = ((lunarMonth − hourZhi + 2) % 12) + 1
身宫地支：shenZhi = ((lunarMonth + hourZhi) % 12) + 1
```

**推导**（寅上起正月）：
- 寅宫(3)起正月 → monthStart = lunarMonth + 2
- 命宫：从 monthStart 逆数至生时 → mingZhi = monthStart − (hourZhi − 1) = lunarMonth − hourZhi + 3 → 归一化为 `((lunarMonth − hourZhi + 2) % 12) + 1`
- 身宫：从 monthStart 顺数至生时 → shenZhi = monthStart + (hourZhi − 1) = lunarMonth + hourZhi + 1 → 归一化为 `((lunarMonth + hourZhi) % 12) + 1`

**十二宫排列**：从命宫起逆时针依次排布：

```
palaceZhi[i] = ((mingZhi − 1 − i + 12) % 12) + 1    i=0..11
```

| i | 宫名 | 地支偏移 |
|---|------|----------|
| 0 | 命宫 | mingZhi |
| 1 | 兄弟 | −1 |
| 2 | 夫妻 | −2 |
| 3 | 子女 | −3 |
| 4 | 财帛 | −4 |
| 5 | 疾厄 | −5 |
| 6 | 迁移 | −6（对冲） |
| 7 | 交友 | −7 |
| 8 | 官禄 | −8 |
| 9 | 田宅 | −9 |
| 10 | 福德 | −10 |
| 11 | 父母 | −11 |

**验证**：正月+子时 → 命宫寅(3)；正月+丑时 → 命宫丑(2)

## 0.2 十二宫天干（五虎遁）

**输入**：`yearGan Gan`（1-10），`mingZhi Zhi`（1-12）

**虎遁口诀**：

| 年干 | 寅上起 | yinGan |
|------|--------|--------|
| 甲(1)、己(6) | 丙 | 3 |
| 乙(2)、庚(7) | 戊 | 5 |
| 丙(3)、辛(8) | 庚 | 7 |
| 丁(4)、壬(9) | 壬 | 9 |
| 戊(5)、癸(10) | 甲 | 1 |

**公式**：`yinGan = ((yearGan − 1) % 5) × 2 + 3`，归一到 1–10

**命宫天干**：`mingGan = yinGan + (mingZhi − 3)`，归一到 1–10

**十二宫天干**（随地支而行）：

```
palaceGan[i] = ((mingGan − 1 − i + 10) % 10) + 1    i=0..11
```

| 五行 | 局数 | 局名 |

## 0.4 紫微星定位

**输入**：`juShu int`（2/3/4/5/6），`lunarDay int`（1-30）

### 算法

1. 算组序：`n = ceil(lunarDay / juShu)`，即 `(lunarDay + juShu − 1) / juShu` 整除
2. 紫微从各局起始宫逆数 n 位（含起始宫）：

```
ziweiPos = (startPos − (n − 1)) mod 12    // 0-based
```

### 各局起始宫

| 水二局(2) | 寅 | 2 |

| 土五局(5) | 午 | 6 |
| 火六局(6) | 酉 | 9 |

## 0.5 十四主星安星

**输入**：`ziweiPos int`（紫微所在宫位 0–11）

### 紫微系 6 星

| 星名 | 简称 | 相对紫微偏移 |
|------|------|-------------|
| 紫微 | 紫 | 0 |
| 天机 | 机 | −1 |
| 太阳 | 阳 | −3 |
| 武曲 | 武 | −4 |
| 天同 | 同 | −5 |
| 廉贞 | 廉 | −8 |

紫微系逆排，偏移 −2/−6/−7 位为空宫，不计入安星。

### 天府星定位

天府与紫微一一对应：`tianfuPos = (ziweiPos + 2) % 12`

验证：紫微在寅(2)→天府在辰(4)；紫微在子(0)→天府在寅(2)；紫微在辰(4)→天府在午(6)。

### 天府系 8 星

| 星名 | 简称 | 相对天府偏移 |
|------|------|-------------|
| 天府 | 府 | 0 |
| 太阴 | 阴 | +1（顺1） |
| 贪狼 | 贪 | +2（顺2） |
| 巨门 | 巨 | +3（顺3） |
| 天相 | 相 | +4（顺4） |
| 天梁 | 梁 | +5（顺5） |
| 七杀 | 杀 | +6（顺6） |
| 破军 | 破 | +10（顺10） |

### 安星规则

紫微系逆排，天府系顺排，位置归一到 0–11：

```
紫微系_i = (ziweiPos − offset_i + 12) % 12
天府系_i = (tianfuPos + offset_i) % 12
```

**输出**：`map[StarIndex]int{starName: palaceIndex}`

## 0.6 辅星安星

13 颗辅星，各依赖一种干支输入。所有公式输出 0-based palace index。

### 按生年天干（yearGan 1–10）

**禄存**：年干定禄存宫（十干禄位）

| yearGan | 甲1 | 乙2 | 丙3 | 丁4 | 戊5 | 己6 | 庚7 | 辛8 | 壬9 | 癸10 |
|---------|-----|-----|-----|-----|-----|-----|-----|-----|-----|------|
| 禄存(地支) | 寅 | 卯 | 巳 | 午 | 巳 | 午 | 申 | 酉 | 亥 | 子 |
| 宫位 index | 2 | 3 | 5 | 6 | 5 | 6 | 8 | 9 | 11 | 0 |

可用表驱动：`lucunTable[10] = {2,3,5,6,5,6,8,9,11,0}`，下标 `yearGan-1`。

**天魁**：按年干（甲戊庚→丑，乙己→子，丙→亥，丁→酉，辛→午，壬→卯，癸→巳）

| yearGan | 甲1 | 乙2 | 丙3 | 丁4 | 戊5 | 己6 | 庚7 | 辛8 | 壬9 | 癸10 |
|---------|-----|-----|-----|-----|-----|-----|-----|-----|-----|------|
| 天魁 | 丑(1) | 子(0) | 亥(11) | 酉(9) | 丑(1) | 子(0) | 丑(1) | 午(6) | 卯(3) | 巳(5) |

表：`tiankuiTable[10] = {1,0,11,9,1,0,1,6,3,5}`

**天钺**：天魁的对宫

```
tianyueIndex = (tiankuiIndex + 6) % 12
```

### 按生年年支（yearZhi 1–12）

| 星名 | 公式（1-based 输入，0-based 输出） |
|------|-------------------------------------|
| 擎羊 | `qingyangIndex = (yearZhi % 12)` （年支顺数1位） |
| 陀罗 | `tuoluoIndex = ((yearZhi - 2 + 12) % 12)` （年支逆数1位） |
| 天马 | 寅午戌→申(8, 0-based 7)；申子辰→寅(2, 0-based 1)；巳酉丑→亥(12, 0-based 11)；亥卯未→巳(5, 0-based 4) |

擎羊验证：年支子(1)→擎羊丑(1,0-based)；年支亥(12)→擎羊子(11→0)。

天马表（用 yearZhi 直接查）：

| yearZhi | 子 | 丑 | 寅 | 卯 | 辰 | 巳 | 午 | 未 | 申 | 酉 | 戌 | 亥 |
|---------|----|----|----|----|----|----|----|----|----|----|----|----|
| 天马index | 2 | 11 | 8 | 5 | 2 | 11 | 8 | 5 | 2 | 11 | 8 | 5 |

即：`tianmaTable[12] = {7,7,7,4,7,11,7,4,1,11,7,4}`，下标 `yearZhi-1`。

### 按出生月支（lunarMonth 1–12）

| 星名 | 公式（0-based 输出） |
|------|---------------------|
| 左辅 | `zuofuIndex = (lunarMonth + 2) % 12` — 辰起正月，顺数 |
| 右弼 | `youbiIndex = ((11 − lunarMonth + 12) % 12)` — 戌起正月，逆数 |

验证：正月→左辅辰(4→3,0-based) `(1+2)%12=3` ✓，右弼戌(10) `(11-1+12)%12=22%12=10` ✓

### 按出生时支（hourZhi 1–12）

| 星名 | 公式（0-based 输出） |
|------|---------------------|
| 文昌 | `wenchangIndex = ((11 − hourZhi + 12) % 12)` — 戌起子时，逆数 |
| 文曲 | `wenquIndex = (hourZhi + 3) % 12` — 辰起子时，顺数 |
| 地空 | `dikongIndex = ((12 − hourZhi + 12) % 12)` — 亥起子时，逆数 |
| 地劫 | `dijieIndex = (hourZhi + 10) % 12` — 亥起子时，顺数 |

验证：子时→文昌 `(11-1+12)%12=22%12=10`=戌 ✓，文曲 `(1+3)%12=4`=辰(4,0-based) ✓
子时→地空 `(12-1+12)%12=23%12=11`=亥 ✓，地劫 `(1+10)%12=11`=亥 ✓

## 0.9 大限

**输入**：`gender Gender`（male/female），`juShu int`（2/3/4/5/6），`mingGan Gan`

### 顺逆行判定

| 性别 | 生年天干 | 方向 |
|------|---------|------|
| 男 | 甲丙戊庚壬（阳干） | 顺行 |
| 男 | 乙丁己辛癸（阴干） | 逆行 |
| 女 | 甲丙戊庚壬（阳干） | 逆行 |
| 女 | 乙丁己辛癸（阴干） | 顺行 |

即：`forward = (isMale ∧ isYangGan) ∨ (isFemale ∧ ¬isYangGan)`

### 起运岁数

起运岁数 = `juShu`（水2/木3/金4/土5/火6）。无四舍五入调整（简化），直接用局数。

### 大限宫位

从命宫(0)开始，顺行或逆行，每宫十年：

```
顺：daXian[0]=命宫(0-9岁), daXian[1]=兄弟(10-19), daXian[2]=夫妻(20-29), …
逆：daXian[0]=命宫(0-9岁), daXian[1]=父母(10-19), daXian[2]=福德(20-29), …
```

宫位序列即 0.1 的 Palaces 顺序（逆时针），顺行按正序走，逆行按逆序走：
```
顺：0→1→2→…→11
逆：0→11→10→…→1
```

**输出**：`[]DaXianStep{startAge, endAge, palaceIndex}`

## 0.10 流年

**输入**：`liuYear int`，`chart Chart`

### 流年命宫

流年岁数从虚岁起算，由生年 + currentYear + 1 得虚岁，流年命宫 = 从命宫按虚岁数顺数（每岁走一宫）：

```
xuSui = liuYear − birthYear + 1
liuNianMingGongIdx = (xuSui − 1) % 12      // 相对于命宫的偏移（顺行）
```

### 流年主星

流年天干定四化（同 0.7 规则，但用流年天干），四化星仅在流年盘中标记。流年命盘不重新安14主星——主星来自原命盘的流年宫位透视。

### 流年辅星

- 流年擎羊/陀罗：按流年地支规则（同 0.6）
- 流年火星/铃星：按流年地支 × 流年宫位（规则同 0.6）

**输出**：`LiuNianResult{liuNianMingGongIdx, siHua, minorStars}`
