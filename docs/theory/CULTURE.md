# Fivefold Types — 文化语境

> **模型站在多个传统的交界处——中国的五行哲学、西方的心理测量学、本土心理学的關係取向。意识到这些来源，比假装从零开始更诚实。**

本文档陈述 Fivefold Types 的文化定位、方法论边界、与现存理论的关系，以及跨文化考量。理论定义见 [THEORY](THEORY.md)。

---

## 1. 五行是来源，不是枷锁

Fivefold Types 的代数核心（S、C 矩阵、d 向量、ReLU 截断）可以完全脱离"木火土金水"的中文名称运行——以五个符号定义同样的生克结构和分类逻辑。模型做了两层工作：

| 层 | 内容 | 可分离性 |
|------|------|:--:|
| 代数结构 | S、C 置换矩阵、Σ=0 守恒、ReLU、内积分类 | 完全可分离，文化无关 |
| 文化叙事 | 元素命名、角色映射、节气月、类型标签 | 绑定东亚文化符号系统 |

两层之间的关系是**翻译**，不是**等同**。五行生克是 2200 年历史的文化理论，以哲学叙事描述人与自然的关系。Fivefold Types 从中提取了一套代数结构——五行专家称之为"对五行理论最干净的数学翻译"，但这个翻译不等同于五行本身。五行理论中无法编码的部分（反生反克、"有余而往不足随之"的动态平衡智慧、干支历法的时空调度）不在本模型范围内。

这种"翻译"姿态意味着：模型不声称自己是"科学化的五行"，而是"受五行启发的可计算人格-关系模型"。前者是对传统的僭越，后者是对传统的承认。

## 2. 方法论边界

### 2.1 元素-人格映射是工作假设

元素-人格映射——W=开创者、F=发光者、E=培育者、M=规范者、R=蕴蓄者——是试探性工作假设（working hypothesis），非经验验证结论。五行维度与 Big Five（OCEAN）是否同构、以及在非东亚文化中的效度，需跨文化因子分析验证。在获得实证支持前，元素描述应被视为对五行经典语义的现代化转译，而非普适人格结构。

### 2.2 零参数意味着可证伪

系统中所有可调量——S、C、原型向量、solar month table——都是硬编码的。无自由参数意味着所有预测可证伪。可证伪性声明见 [NOVELTY](NOVELTY.md)。

### 2.3 描述不评判

力流分析（Bond）不判好坏——推力和拉力是力的方向，不是关系质量。这种价值中立姿态既是科学品格，也是跨文化使用的前提——一种文化中的"好关系"在另一种文化中可能是"窒息的关系"。力学描述留给用户自己判断。

## 3. 与现存人格理论的关系

Fivefold Types 在人格模型谱系中的位置：

- **不是 Big Five 的替代。** 五行与 OCEAN 可能重叠、可能正交——需要因子分析回答。如果重叠，Fivefold Types 的增量贡献是 Bond——Big Five 描述个体差异，Fivefold Types 计算关系动力。
- **不是 MBTI 的竞争者。** MBTI 是离散类型 + 认知功能栈。Fivefold Types 是连续维度 + 生克关系网络。分类标签只是传播层，不是理论核心。
- **不是"东方的"人格模型。** 这个标签暗示模型只在东亚文化中有效。代数结构是文化无关的——一个从未听说过"木"的人也可以有 d = [0.3, -0.1, ...] 并在 Bond 中得到有意义的 Δ。文化内容（元素命名、节气月）是可选的文化层，可替换而核心引擎不变。

与现存系统的详细对照见 [NOVELTY](NOVELTY.md)。

## 4. 自我概念的两种语言

当代人格心理学的主流用语偏向**独立型自我**（independent self）——"你的类型""你的特质""你是谁"。这是西方心理学在西方文化中产生的自然用语。但五行思想的深层逻辑更接近**互依型自我**（interdependent self）——一个人不是固定的，而是在与他人的关系中被持续定义的。

Fivefold Types 的数学结构同时容纳了这两种语言：

- **独立型视角**：d 是"你的形状"——一份稳定的行为倾向剖面
- **互依型视角**：d_eff = d + Δ^B——你在与 B 的关系中有效形状不同。木偏盛的人推高你的火，金偏盛的人拉低你的木。同一个人，在不同的关系中是不同的

后者在华人本土心理学中有深厚的理论对应。杨国枢（2004）的"關係取向"理论指出，中国人的社会行为以关系而非个体特质为基本单位。黄光国的"人情与面子"模型描述了不同关系类型中的差异化自我呈现。Fivefold Types 的 Δ 公式无意中为这种关系取向提供了一个形式化工具——一个人在不同关系中 d_eff 不同，不是"面具"或"不一致"，而是关系的构成性特征。

跨文化的比较视野也值得提及：
- **印度三特质**（Sattva/Rajas/Tamas）：与五行同样以元素-特质映射描述人格，但关注焦点是意识的澄清度而非对外功能的偏向。两者的比较分析可能揭示五行维度的独特性。
- **非洲 Ubuntu**（"我在因为我们在"）：强调共同体人格——个体只能在社群中定义。Fivefold Types 的他评系统（peer assessment + peers_aggregated）与此有隐性共鸣——"你是你，也是别人眼中的你"。

## 5. 跨文化考量

### 5.1 节气月的文化范围

十二节气月源于东亚季风气候区的农耕历法。solar month table 的季节-元素对应（如春木、夏火、秋金、冬水）反映北半球温带季风节律，非全球自然法则。

- **南半球**：天文节气月判定全球统一（太阳黄经 315°=立春），但季节相位相反。南半球用户将节气月索引 +6 翻转——使季节文化层匹配当地气候，而非修正五行理论。核心引擎（S、C 矩阵、ReLU、分类）不含季节依赖，全球统一。
- **赤道地区**：无四季节律，关闭 Flow 功能（d_eff = d）。叠加 solar month table 在此类地区是无意义的信号注入。

Feng Shui 社区对此存在两派争议——适应派主张季节-元素映射应随气候翻转，普遍派（如 Heluo, 2002）主张五行本质为地磁而非太阳位置、全球不变。本模型取中间路线：核心代数不变，季节文化层本地化。

节气月在本模型中定位为"基于东亚传统的季节文化提示"，不应被用户理解为自然定律。

### 5.2 Ipsative 数据的跨文化解释

三元追选（ipsative）格式在不同文化中有不同的反应模式。每题 3 选 2（排除 1 个）以排除判断代替选择判断——排除"最不像"比选择"最像"更少受社会赞许性驱动，对谦虚偏差的文化群体更友好。高语境文化被试倾向于平衡选择而非极端排除，集体主义文化中社会赞许性可能推高社群认可的特质（如培育者、规范者）。这些偏差影响跨文化常模比较——但 Fivefold Types 的设计目标是个体内部分析和双人关系计算，而非常模排名。公开用户浏览页已标注 ipsative 数据的个体间比较局限。

### 5.3 元素命名的文化可译性

元素名在跨文化传播中存在语义偏移风险——"Fire"在某些文化中可能关联破坏而非温暖，"Metal"可能关联冷硬而非精炼。角色标签（开创者、发光者等）是对元素语义的行为化翻译，降低了文化特定联想的干扰。后续增加新 locale 时应独立验证标签系统在该文化中的效度，而非简单翻译英文版本。

### 5.4 实证状态

五行人格理论的跨文化验证尚属空白。Hsu (2006) 基于《人物志》和阴阳五行开发了中文人格量表并在台湾大学生中验证了信效度，但无跨文化复制研究。Yoyong (2025) 的系统文献回顾指出 Wu Xing 人格模型"实证上探索不足，需要被操作化为可测量的行为指标"——这正是 Fivefold Types 试图做的事：将五行从哲学叙事转化为可计算、可证伪的行为模型。节气月-人格的跨半球实证检验目前无任何已发表研究。

**跨文化验证的优先级**：数据积累至跨文化样本（每组 N ≥ 200）后，应进行项目水平的 DIF（differential item functioning）分析——检测哪些具体选项在不同文化中系统性偏离其元素映射。在获得 DIF 证据前，所有选项的 W/F/E/M/R 映射应被视为可被数据修正的工作假设。DIF 结果将直接指导后续版本的选项修订和跨文化适配。

---

## 6. 立场总结

本模型将中国五行生克理论翻译为代数结构，保留生克关系的不对称性和方向性。代数核心文化无关，文化叙事可替换。元素-人格映射是工作假设。"独立型自我"和"互依型自我"两种视角并存——d 刻画个体，Δ 刻画关系。模型不声称自己是"东方的"或"科学的"，只声称自己是可计算、可证伪的。
