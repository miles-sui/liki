## 五、四柱推算

### 5.1 年柱

**公式**：

```
年干 g = (year - 3) % 10     // 0 → 癸(10), 1 → 甲, ...
年支 z = (year - 3) % 12     // 0 → 亥(12), 1 → 子, ...
```

**关键边界**：以**立春**为年界。立春前（约 2 月 3-5 日）属上一年。

**验证**：2024 年 → (2024-3)%10=1 甲，(2024-3)%12=5 辰 → **甲辰年**。

### 5.2 月柱

**月支 —— 固定，以节气为界**：

| 月序 | 月支 | 节气区间 |
|------|------|----------|
| 正月 (1) | 寅 | 立春 → 惊蛰 |
| 二月 (2) | 卯 | 惊蛰 → 清明 |
| 三月 (3) | 辰 | 清明 → 立夏 |
| 四月 (4) | 巳 | 立夏 → 芒种 |
| 五月 (5) | 午 | 芒种 → 小暑 |
| 六月 (6) | 未 | 小暑 → 立秋 |
| 七月 (7) | 申 | 立秋 → 白露 |
| 八月 (8) | 酉 | 白露 → 寒露 |
| 九月 (9) | 戌 | 寒露 → 立冬 |
| 十月 (10) | 亥 | 立冬 → 大雪 |
| 十一月 (11) | 子 | 大雪 → 小寒 |
| 十二月 (12) | 丑 | 小寒 → 立春 |

**月天干公式（五虎遁）**：`g_month = (g_year × 2 + month_num) % 10`

**口诀**：甲己之年丙作首，乙庚之岁戊为头；丙辛必定寻庚起，丁壬壬位顺行流；戊癸何方觅，甲寅之上好追求。

| 年干 | 正月起 |
|------|--------|
| 甲、己 | 丙寅 |
| 乙、庚 | 戊寅 |
| 丙、辛 | 庚寅 |
| 丁、壬 | 壬寅 |
| 戊、癸 | 甲寅 |

### 5.3 日柱

采用**儒略日法**（不限年份范围，支持 1900 前后）。

**儒略日计算**：

```python
def julian_day(y, m, d):
    if m <= 2:
        y -= 1
        m += 12
    A = y // 100
    B = 2 - A + A // 4
    return int(365.25*(y+4716)) + int(30.6001*(m+1)) + d + B - 1524
```

**基准日**：1900 年 1 月 1 日 = 甲戌（第 10 位，0-based=9）

**推算**：

```
diff_days = 儒略日(目标日期) - 儒略日(1900,1,1)
gz_index  = (9 + diff_days) % 60       # 六十甲子中的位置
日干       = gz_index % 10              # 0=甲 ... 9=癸
日支       = gz_index % 12              # 0=子 ... 11=亥
```

**关键边界**：日柱以**子时（23:00）** 为界。23:00 后计为次日。

### 6.2 太阳黄经与节气算法

八字排盘使用**节**（十二个 major solar terms）划分月柱，使用**气**（十二个 minor solar terms）辅助大运计算。

节气通过太阳黄经（ecliptic longitude）确定，非农历固定日期。太阳在黄道上每运行 15°为一个节气。

**24 节气黄经**：

| 节 (major term) | 黄经 | | 气 (minor term) | 黄经 |
|-----------------|------|---|-----------------|------|
| 立春 | 315° | | 雨水 | 330° |
| 惊蛰 | 345° | | 春分 | 0° |
| 清明 | 15° | | 谷雨 | 30° |
| 立夏 | 45° | | 小满 | 60° |
| 芒种 | 75° | | 夏至 | 90° |
| 小暑 | 105° | | 大暑 | 120° |
| 立秋 | 135° | | 处暑 | 150° |
| 白露 | 165° | | 秋分 | 180° |
| 寒露 | 195° | | 霜降 | 210° |
| 立冬 | 225° | | 小雪 | 240° |
| 大雪 | 255° | | 冬至 | 270° |
| 小寒 | 285° | | 大寒 | 300° |

**天文算法**（项目现有 `internal/tianwen/flow.go`）：

1. **计算指定时刻的太阳黄经** — 基于平均轨道要素 + 地球轨道偏心率修正（Keplerian 三阶谐波）：

```go
func solarLongitude(t time.Time) float64:
    jd := julianDay(t)
    T := (jd - 2451545.0) / 36525.0       // 儒略世纪数
    L0 := 280.46646 + 36000.76983*T       // 太阳平黄经
    M := 357.52911 + 35999.05029*T        // 太阳平近点角
    C := (1.914602 - 0.004817*T)*sin(M)   // 中心差（椭圆轨道修正）
       + (0.019993 - 0.000101*T)*sin(2M)
       + 0.000289*sin(3M)
    return L0 + C  // 真黄经
```

2. **用 Newton 迭代法反推太阳到达某一黄经的具体时刻**：

```go
func solarTermDate(year int, targetLon float64) time.Time:
    t := approximate_date          // 粗估: 1月1日 + 约15天/节气
    for iter < 20:
        lon := solarLongitude(t)
        diff := targetLon - lon    // 若>180°，修正为短弧路径
        if |diff| < 0.01: break    // 精度 < 0.01°
        t += diff / 0.9856 * 1day  // 太阳每日约走 0.9856°
    return t
```

**精度**：< 0.01°（约 15 分钟），对八字时辰判断足够。

**现有代码**：`internal/tianwen/flow.go` 中 `solarLongitude()`、`solarTermDate()`、`PrecomputeSolarTerms()` 可直接复用。八字排盘需额外调用 `solarTermDate(year, 315)` 获取立春日期确定年柱边界。

### 6.3 真太阳时公式

$$AST = LST - DS + 4 \times (L_{local} - L_{std}) + EoT$$

| 参数 | 含义 |
|------|------|
| AST | 真太阳时（分钟，相对 00:00） |
| LST | 当地标准时间（分钟） |
| DS | 夏令时修正（60 分钟或 0） |
| L_local | 出生地经度（度，东经为正） |
| L_std | 标准时区中央经度（北京时间 = 120°E） |
| EoT | 均时差（分钟） |

### 6.2 Spencer 均时差公式

$$B = \frac{360 \times (N - 81)}{365} \quad\text{（度）}$$

$$EoT = 9.87\sin(2B) - 7.53\cos(B) - 1.5\sin(B) \quad\text{（分钟）}$$

其中 N 为日序数 (1 = 1 月 1 日)。

### 6.3 关键数值

| 日期 | EoT | 含义 |
|------|-----|------|
| 2 月 11 日 | −14.1 分 | 真太阳时最慢 |
| 11 月 3 日 | +16.4 分 | 真太阳时最快 |
| 4 月 15 日 | ≈0 | 两时一致 |
| 6 月 13 日 | ≈0 | 两时一致 |
| 9 月 1 日 | ≈0 | 两时一致 |
| 12 月 25 日 | ≈0 | 两时一致 |

全年变化范围约 ±16 分钟，曲线呈"8"字形（Analemma）。

## 六之附录：南半球考量

### 节气是天文现象

二十四节气由太阳黄经定义，是**全球同步的天文事件**。立春（315°）在全球同一 UTC 时刻发生，不受南北半球影响。

### 关键区分

| 维度 | 北半球 | 南半球 | 八字处理 |
|------|--------|--------|----------|
| 节气时刻 | 立春 ≈ 2月4日 | 立春 ≈ 2月4日（同一时刻） | 无差异 |
| 气象季节 | 2月=冬末 | 2月=夏末 | 无关 |
| 真太阳时 | 经度修正 + EoT | 经度修正 + EoT（同公式） | 无差异 |
| 月柱 | 寅月=立春起 | 寅月=立春起（天文一致） | 无差异 |
| 日柱 | 按公历日期+23:00子时 | 同左 | 无差异 |

### 两种流派

八字南半球排盘存在两种观点：

1. **天文派（主流）**：八字基于天文历法，非气象季节。太阳黄经决定月柱，全球统一。寅月就是立春到惊蛰，无论在哪个半球。**本项目默认此派。**

2. **对称派（少数）**：主张南半球月柱应翻转 6 个月（如南半球 2 月用申月代替寅月），以使五行旺衰与当地气候对应。

### 本项目立场：天文为体，物候为用

八字与 Flow 区别对待：

| | 八字排盘 | Flow（季节性流动） |
|---|---|---|
| 性质 | 天道——太阳周期 | 地道——地方物候 |
| 南半球 | 不翻，全球统一 | 翻 6 个月 |
| 理由 | 八字是封闭历法体系，内洽优先 | Flow 反映当下环境五行，应与当地季节一致 |

**八字排盘**采用天文派——不翻转。八字体系根植于太阳黄经，非地方气候，同一 UTC 时刻全球排盘结果一致。

**Flow 计算**则反向——南半球用户应根据当地实际季节翻转月支。寅月在南半球对应夏末秋初，应表达火旺金生而非木旺火生。Flow 描述的是人当下所处的环境能量状态，须与当地物候对齐。

如需对称派八字排盘，可在请求中传入 `hemisphere: "southern"` 参数，作为可选实验功能。

### 7.4 大运

**起运规则**：

| 性别 | 年干 | 排法 |
|------|------|------|
| 男 | 阳（甲丙戊庚壬） | 顺排（月柱 → 下一柱） |
| 男 | 阴（乙丁己辛癸） | 逆排 |
| 女 | 阳 | 逆排 |
| 女 | 阴 | 顺排 |

**起运年龄**：出生日到下一个（顺排时）或上一个（逆排时）"节"的天数 ÷ 3（3 天 = 1 岁，余 1 天 = 4 个月）。

**每步大运**：10 年一柱，通常排 8 步（80 年）。

大运天干：顺排取月柱的下一组干支，逆排取上一组。
