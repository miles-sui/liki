# Fivefold Types — 创新声明

> **Fivefold Types 是一个基于五行生克代数的零参数人格-关系模型。偏盛者发力，偏弱者受力。**

---

## 与现有系统的对照与创新

### 关系计算

Big Five 提供统计正交的五个连续维度，有学术合法性，但**没有两个剖面之间的原生互动算术**——知道两个人的 OCEAN 分数，算不出他们之间的动力。MBTI 的 16 型将配对视为叙事，Enneagram 的类型动力学同样不可计算。八字有十神系统描述人与外界的关系，但依赖出生日期。

Fivefold Types 用两个置换矩阵（S、C）承载全部生克运算。`d_eff = d + Sᵀd⁺ - Cᵀd⁺`——连续剖面之间的互动完全由线性代数表达。五行维度之间的非正交性不是统计缺陷，是理论资产——正是生克结构使关系运算成为可能。ReLU 将"偏盛者发力"落实为精确数学表达：只有自身有余力的元素才对外输出影响。

### 参数化

Big Five 依赖因子载荷从样本估计，Enneagram 有翼和三元组作为附加结构。这些系统都引入了可调成分。

Fivefold Types **零参数**。S、C、原型向量、solar month table 全部硬编码。生克力直接等于源强——偏盛者输出多少，就推动/约束多少。没有为"调出合理数字"而设的自由参数，模型行为完全由 d 的实测值决定。

### 输出形式

MBTI 给出离散的四字母标签，Enneagram 给出类型编号和大类归属。Big Five 是连续剖面但无法做关系计算。这些系统的共同点是**用单一标签或分数描述一个人**。

Fivefold Types 的 Bond 输出**力流描述**而非"配/不配"的单一分数——交互强度、张力密度、主要生克力通道。用户看到的是"力在怎么走"——B 在推高什么、在约束什么——而不被一个数字替自己做判断。内部运算用 d（Σd=0，生克守恒），对外展示用 p（Σp=1），代数与语义各得其所。

### 分类体系

MBTI 用四个二分维度产生 16 种离散类型，Enneagram 有 9 型加翼。

Fivefold Types 的 25 个原型锚定于单纯形棱上的固定向量（d-空间，Σd=0）——5 个纯型在顶点，20 个复合型由统一参数 α=0.52 定位在棱上。α 由 Monte Carlo 在 5-单纯形上均匀采样优化确定，使 25 型分布接近均匀。分类路径为 d → cos(d, d_proto)，平局取 d 最大值元素。标签锚定于固定向量，可稳定描述和比较。

### 时间维度

传统八字通过干支历法和出生日期推算大运流年，是宿命模型。

Fivefold Types 用 solar month table 的 12 个 one-hot 向量叠加 d——个性化的能量气候历。不需要干支历法，不需要出生日期。从宿命模型变为自评模型的提示工具。

## 可证伪性

连续维度模型支持标准的心理测量学验证：

- **重测信度**：d 向量相关系数
- **结构效度**：生克结构（S、C）是模型的输入公理，非待验证假设。结构效度验证的是以此结构导出的预测——Bond 力流方向是否与关系量表在预测方向上一致，Flow 季节效应是否与自评感受吻合
- **区分效度**：d 剖面与 Big Five 是否测量不同构念
- **生态效度**：Flow 年历与自评季节感受是否一致
- **会聚效度**：Bond 力流分量与关系量表是否在预测方向上相关
