# Fivefold Types — 编码规范

## 状态向量

### 内部表示：d（形状）

```
d = [d_W, d_F, d_E, d_M, d_R]

Σd_i = 0
中性点: d = [0, 0, 0, 0, 0]
```

d_i > 0 → 偏盛，d_i < 0 → 偏弱。

### 对外展示：p（比例）

p 由 raw 直接归一化得出（Σp=1），仅用于展示。d 和 p 同源于 raw，满足 p = 0.2 + d/5——同源的代数恒等式，非互推。内部运算始终用 d。

### 偏盛掩码：d⁺

```
d⁺_i = max(d_i, 0)
```

只有偏盛的元素参与生克力输出。偏弱元素的生克力为零。

### JSON

```json
{
  "d": { "wood": 0.150, "fire": 0.100, "earth": -0.050, "metal": -0.100, "water": -0.100 },
  "p": { "wood": 0.230, "fire": 0.220, "earth": 0.190, "metal": 0.180, "water": 0.180 }
}
```

d 和 p 同源于 raw，数值满足 p = 0.2 + d/5。

### S 与 C 矩阵

元素索引序: W=0, F=1, E=2, M=3, R=4。

生矩阵 S (5×5), S[j][k]=1 ⇔ 元素 j 生元素 k:

```
       被生
        W  F  E  M  R
生  W [ 0  1  0  0  0 ]    木生火
    F [ 0  0  1  0  0 ]    火生土
    E [ 0  0  0  1  0 ]    土生金
    M [ 0  0  0  0  1 ]    金生水
    R [ 1  0  0  0  0 ]    水生木
```

克矩阵 C (5×5), C[j][k]=1 ⇔ 元素 j 克元素 k:

```
       被克
        W  F  E  M  R
克  W [ 0  0  1  0  0 ]    木克土
    F [ 0  0  0  1  0 ]    火克金
    E [ 0  0  0  0  1 ]    土克水
    M [ 1  0  0  0  0 ]    金克木
    R [ 0  1  0  0  0 ]    水克火
```

S 与 C 均为置换矩阵。保持向量和：Σ(Sx)=Σ(x), Σ(Sᵀx)=Σ(x)。

## 身份编码

### 分类算法

Profile 和 prototype 统一在单纯形 d-空间（Σd=0）。分类使用余弦相似度——方向匹配，不受幅度影响：

```
identity = argmax_{k}  cos(d, d_protoₖ)
         = argmax_{k}  d · d_protoₖ / (‖d‖ · ‖d_protoₖ‖)
```

不需要 ReLU。余弦相似度自然分离了"形状方向"和"形状强度"——纯型的 d 模长（≈0.894）天然大于复合型（≈0.552），点积会偏袒纯型；余弦消除了这种幅度偏袒。

原型 d_protoₖ 存储在单纯形 d-空间（Σd=0）。纯型 d_proto = [0.8, -0.2, -0.2, -0.2, -0.2]（对应 p=[1,0,0,0,0]）。20 个复合型共享统一参数 **α**：d_proto = [α-0.2, 1-α-0.2, -0.2, -0.2, -0.2]（对应 p=[α, 1-α, 0, 0, 0]）。α 控制复合原型在单纯形棱上的位置。

平局时选主导元素（首字母）在 d 中取值最大者。平局仅发生在 Voronoi 边界（测度零集），对分布无统计影响。

### 为什么是 α=0.52

纯型在 5-单纯形的顶点上，每个顶点连接 4 条棱。复合型在棱上，每条棱有 2 个方向。分类器的 Voronoi 划分取决于原型在棱上的位置参数 α：

- α → 0.5：原型趋近棱中点，纯型 Voronoi 区域被压缩
- α → 1.0：原型趋近顶点，纯型和同方向复合型趋同

α 由 Monte Carlo 在 triad 追选模拟下确定。目标：纯型占比合理（不过半也不可忽略），元素层面完全平衡。

```
α=0.50: 纯型 ~23%  复合均匀比 2.3x
α=0.52: 纯型 ~19%  复合均匀比 2.2x  元素平衡 ±0.6%  ← 选定
α=0.55: 纯型 ~17%  复合均匀比 2.2x
α=0.60: 纯型 ~16%  复合均匀比 2.2x
```

选定 α=0.52。纯型 ~19%（约 1/5），复合型 25 类在 5 个几何组内均匀分布，元素平衡完美（W/F/E/M/R 各 ~20.0%）。均匀单纯形采样（Dirichlet）下 S₅ 对称性保证各元素 Voronoi 细胞等体积——实证验证通过。

阈值附近分类对微小波动敏感。d ≈ 0 时余弦分数的分子分母均趋于零，此时选 d 中最大元素对应的纯型。这是连续性中的退化情况——ipsative 格式下实际几乎不发生。

## 可证伪性

α=0.52 是模型唯一的自由设计参数（S、C 矩阵是定义，cos(d) 是设计选择）。零参数纪律要求 α 的选择可被数据质疑。

### 可证伪预测

选取 α=0.52 的隐含预测：(1) 纯型占比约 19%（~1/5）；(2) 五个主导元素各约 20.0%，元素平衡在 ±1% 以内；(3) 25 型在 5 个几何组内均匀分布。

### 证伪条件

若大样本（N > 10,000）中出现以下任一情况，且排除混淆因子后偏差持续存在：

1. 纯型占比 < 14% 或 > 25%
2. 任一主导元素占比 < 17% 或 > 23%
3. 单一复合型频度 < 0.5% 或 > 6%

混淆因子同前：问卷措辞偏差、样本人口学偏倚、Ipsative 格式的端点效应。

则 α=0.52 被证伪。

### 修正策略

α 是模型定义参数，非数据集拟合参数。改变 α 意味着全部 25 型原型向量改变——等同于模型版本变更。修正须满足：

- 至少两个独立研究团队的跨文化数据集独立验证
- 替代 α 的误差 margin 明确（如 0.52 ± 0.02 的置信区间）
- 版本断点明确记录，新旧分类的迁移方案事先定义

在此条件满足前，α=0.52 保持为定义性常量。
