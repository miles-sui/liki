## C. 流月流日流时

### C.1 干支计算

年月日时的干支沿用 `tianwen` 节气方法或六十甲子推算：

```
月干支 = (年天干 × 2 + 月份) mod 甲子
日干支 = 已知基准日 + 距基准日天数 mod 六十甲子
时干支 = 日天干 × 2 + 时辰序 mod 甲子
```

简化：复用或新实现天干地支推算，与八字一致。

### C.2 流月

| 函数 | 输入 | 输出 |
|------|------|------|
| `LiuYueMingGong(lunarMonth, birthYear, liuYear)` | 农历月+生年+流年 | PalaceIndex |
| `LiuYueSiHua(lunarMonth, liuYear)` | 农历月+流年 | SiHuaResult |
| `LiuYueMinors(liuMonthZhi, hourZhi)` | 流月地支+生时时支 | map[star]index |

流月命宫 = 从流年命宫按农历月顺数（正月→1位，二月→2位…）。流月天干用五虎遁推（流年天干定寅上）。

### C.3 流日

| 函数 | 输入 | 输出 |
|------|------|------|
| `LiuRiMingGong(lunarDay, birthYear, liuYear, liuMonth)` | 农历日+生年+流年+流月 | PalaceIndex |
| `LiuRiSiHua(liuDayGan)` | 流日天干 | SiHuaResult |

流日命宫 = 从流月命宫按农历日顺数（初一→1位）。

### C.4 流时（可选，低优先级）

同上模式，从流日命宫按时辰顺数。
