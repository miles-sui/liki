# 名字评分可靠吗

## Direct Answer

名字评分（五格数理）可以参考但不能作为唯一标准。一个数理满分但五行不匹配的名字，每天都在影响你的五行平衡。五行补你的名字如果数理一般，依然可以有效发挥作用。五行优先于数理。

## Key Signals · BaZi 指标

- 数理好 + 五行补 → 最优解。
- 数理好 + 五行克 → 不可取。数理满分也弥补不了五行不匹配的影响。
- 数理一般 + 五行补 → 可以使用。五行的作用力大于数理。
- 数理差 + 五行克 → 赶紧换名字。

## Decision Map

- 五行补 → 数理只是加分项。
- 五行克 → 数理满分也救不了。
- 都可以 → 以五行为第一优先级。

## Why You're Asking

你不是在怀疑评分方法的准确性——是发现不同的方法和师傅给出的结论不一样。五行标准是唯一不变的。
